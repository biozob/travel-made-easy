# TRAVEL MADE EASY

### This is a presentation project.

## Run Locally

Clone the project

```bash
  git clone https://github.com/biozob/travel-made-easy.git
```

Go to the project directory

```bash
  cd travel-made-easy
```

Go to backend folder and rename .env.example to .env

```bash
  mv .env.example .env
```

Install with Docker

```bash
  docker-compose up
```
